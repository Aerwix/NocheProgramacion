# Reconoce Automaticamente Codigo

Reconoce codigo y los agrega a la Web siempre que esten el folde correcto

-   Nombre_Web > Normbre_Folden

-   P5.js > P5
-   Processing > Processing
-   JavaScript > JavaScript
-   Node.js > Node
-   Diagrama > Diagrama
